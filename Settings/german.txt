// ---------- die Einstellungen ----------
settings: Einstellungen

generalsettings: Allgemeine Einstellungen
videoplayersettings: Einstellungen für Videoplayer und Wiedergabelisten
advancedsettings: Erweiterte Einstellungen
informations: Informationen

// Allgemeine Einstellungen
language: Sprache
autoupdates: Überprüfen Sie bei jedem Start, ob ein Update verfügbar ist
manualupdatecheck: Manuelle Aktualisierungsprüfung

//  Einstellungen für Videoplayer und Wiedergabelisten
stretchtofit: Strecken Sie das Video auf den Bildschirm aus
sortvideoslist: Sortiere die Liste der Videos und Playlists
ascending: Aufsteigend
descending: Absteigend

maxdropdownplaylists: Maximale Anzahl an Wiedergabelisten, die gleichzeitig in der Liste angezeigt werden

// Erweiterte Einstellungen
showfiles: Dateien anzeigen
resetfiles: Dateien zurücksetzen

// Informationen
numberplaylists: Anzahl der Wiedergabelisten
weightplaylists: Gewicht der Playlists
refreshinformations: Informationen aktualisieren

// ---------- Nachrichtenbox ----------
newversion: Eine neue Version ist verfügbar, klicken Sie auf YES, um sie herunterzuladen!
nonewversion: Keine neue Version verfügbar!
errorconnection: Ein Verbindungsfehler ist aufgetreten, um festzustellen, ob eine neue Version der Software verfügbar ist. Die Zeitüberschreitung wurde überschritten. Bitte versuchen Sie es später erneut.

deletefiles: Möchten Sie die Software zurücksetzen?
restartsoftware: Bitte starten Sie die Software neu

playlistexist: Dieser playlist existiert schon
videoexist: Dieses Video existiert bereits
selectvideo: Wählen Sie ein Video bitte aus
selectplaylist: Bitte wählen Sie eine Playlist
deletevideo: Möchtest du dieses Video löschen?
deleteplaylist: Möchten Sie diese Playlist löschen?

// ---------- Wiedergabelisten ----------

// Texte
playlists: Wiedergabelisten
playlistslist: Liste von playlists

playlistname: Namen der playlist
videoname: Namen der videos
videodescription: Beschreibung des Videos
cancel: Stornieren

open: Offen

filename: Dateiname
filedescription: Dateibeschreibung
fileduration: Dateidauer

// Kontextmenüleiste
playvideo: Sieh dir das Video an
addvideo: Füge ein Video hinzu
removevideo: Video löschen
editvideo: Video bearbeiten
exportvideo: Video exportieren

addplaylist: Einen playlist hinzufügen
removeplaylist: Den playlist beseitigen
editplaylist: Den playlist ändern
exportplaylist: Den playlist ausführen
